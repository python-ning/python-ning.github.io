解决Ubuntu14.04 kylin LTS下Sublime Text 3无法输入中文的问题

发表于 2015-10-20   |   分类于 linux   |  

在网上找了很多都没能解决这个问题，亲测了这种方法可以解决。

1.下载我们需要的文件，打开终端
输入：

cd ~

git clone https://github.com/lyfeyaj/sublime-text-imfix.git

2.将subl移动到/usr/bin/，并且将sublime-imfix.so移动到/opt/sublime_text/（sublime的安装目录）
终端输入：

cd ~/sublime-text-imfix
sudo cp ./lib/libsublime-imfix.so /opt/sublime_text/
sudo cp ./src/subl /usr/bin/

3.然后打开sublime Text 3 输入中文试试：

LD_PRELOAD=./libsublime-imfix.so subl

4.但是这样子只能用上述命令行启动才能输入中文，我们需要更简单的办法，我选择新建一个shell脚本，很简单，就一句话。
sublime新建文件输入：

#!/bin/bash
LD_PRELOAD=/opt/sublime_text/libsublime-imfix.so subl

5.然后将文件另存为sublime（注意：没有后缀），并且放到/home文件夹。

6.至此，所有工作完成了。想要启动，直接终端输入：bash ~/sublime，就可以启动sublime，并且能够输入中文。

7.如果不需要输入中文，直接按以前的方式打开即可。建脚本文件的目的就是方便打开能输入中文的sublime。
